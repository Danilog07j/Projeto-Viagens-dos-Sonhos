# Viagem dos Sonhos

Um projeto inspirado em plataformas de agências de viagens, desenvolvido com um foco em layout moderno, navegação intuitiva e uma experiência visual atrativa.

## Sobre o Projeto

Esse projeto consiste na recriação visual de um site de agência de viagens, contendo diferentes seções e categorias comuns em plataformas do segmento turístico.

O objetivo principal foi praticar desenvolvimento front-end, organização de componentes e construção de interfaces responsivas, sem o foco em funcionalidades reais de reserva ou integração com APIs.

- Seção de destinos
